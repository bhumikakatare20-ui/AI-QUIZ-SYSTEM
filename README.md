# 🧠 AI Quiz System

A dynamic quiz web application built with Django that lets users test their knowledge across multiple categories. Admins can manage the entire question bank through Django's built-in admin panel — no coding required.

Users can attempt quizzes by category, track their progress question by question, and get a detailed result with score, grade, and answer review at the end.

## Tech Stack
Python · Django · SQLite · HTML · CSS

## Features
- Category-wise quiz selection
- One question at a time with progress bar
- Score, percentage and grade (A–F) on result page
- Detailed answer review with explanations
- Admin panel to add, edit and delete questions
- Custom topic quiz generation without database
