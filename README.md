# AI-QUIZ-SYSTEM
Project Title:AI Based Online Quiz System
Description:
The AI Quiz System is a web-based application developed using Python and Flask. It provides an interactive quiz platform where the difficulty level of questions changes automatically based on the user’s performance. The system starts with easy questions and adjusts to medium or hard depending on whether the user answers correctly or incorrectly.
# Technologies Used:
Python
Flask
HTML
CSS
JavaScript
Session Management
# Features:
Interactive online quiz interface
Automatic difficulty adjustment (Easy → Medium → Hard)
Score calculation system
Random question generation
User-friendly UI design
Result display with final score
